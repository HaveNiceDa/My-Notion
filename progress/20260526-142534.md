## 错误契约

| HTTP | Code | 说明 |
| --- | --- | --- |
| `401` | `UNAUTHORIZED` | 缺失 Bearer token 或 token hash 不存在 |
| `401` | `TOKEN_REVOKED` | PAT 已撤销 |
| `401` | `TOKEN_EXPIRED` | PAT 已过期 |
| `403` | `INSUFFICIENT_SCOPE` | PAT 缺少所需 scope |
| `404` | `NOT_FOUND` | 文档或端点不存在、文档已归档或不属于当前 token 用户 |
| `422` | `VALIDATION_ERROR` | 请求参数或请求体不合法 |
| `429` | `RATE_LIMITED` | 触发固定窗口限流 |
| `500` | `INTERNAL_ERROR` | 未预期服务端错误 |
