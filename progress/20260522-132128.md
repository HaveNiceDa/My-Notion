# 语言与主题二态切换

## 完成的改动

- Web 端语言路由从 `en` / `zh-CN` / `zh-TW` 收敛为 `en` / `zh-CN`。
- 删除共享 i18n 的 `zh-TW.json` 语言包，运行时代码不再引用繁体语言资源。
- Web `LanguageToggle` 从下拉选择改为单击中英互切，并保留当前 path、query、hash。
- Web `ModeToggle` 从下拉选择改为单击浅色/深色互切，移除 `system` 选项。
- Web `ThemeProvider` 默认主题改为 `light`，并设置 `enableSystem={false}`。
- 移动端设置页的语言和主题入口改为单击切换，不再进入二级选择页面。
- 移动端语言支持收敛为 `en` / `zh-CN`，中文设备默认落到 `zh-CN`。
- 移动端主题支持收敛为 `light` / `dark`，移除 `light_blue` / `dark_blue` 配置。
- AI 菜单国际化定义移除 `zh-TW` 字段，并同步更新相关单测。
- 更新 `docs/web-mobile-gap-analysis.md` 中共享 i18n 的语言包说明。
