# 2026-05-27 至 2026-05-31 阶段进展汇总

## 总览

本阶段完成了 My-Notion 从 Web Agent 能力打磨到 CLI/Skills/MCP Agent 生态发布的收口。此前按时间记录的碎片日志已压缩到本文，稳定结论以当前 README、里程碑和 release checklist 为准。

## Web Agent 与 AI 能力

- AI Chat 从独立页面演进为 Web 侧边栏体验，补齐 Markdown 渲染、工具卡片、错误边界和交互细节。
- 后端从单一 RAG pipeline 演进为 ReAct Agent Loop，LLM 可自主调用 `knowledge_search`、`web_search`、`web_extract`、`document_search`、`document_read`、`document_write`、`document_update`、`memory_read`、`memory_write`、`task_plan`。
- RAG 完成 Qdrant + DashScope Embedding 基线、hybrid retrieval、context packing、citation quality 和 Qdrant 离线降级。
- Memory MVP 完成数据模型、确认式写入、语义检索、token/recency fallback、Review UI 与缓存清理。
- 写类工具遵循“预览 -> 用户确认 -> 落库”，默认 dry-run 或 `confirmationRequired`。
- `task_plan` 已作为基础 tool 接入，但完整 Plan 模式仍需补“生成计划 -> 用户确认 -> 步骤执行/状态展示”闭环。

## CLI / Skills / MCP

- CLI 从 `@notion/my-notion-cli` 调整为 npm 包 `@mynotion/cli`，bin 保持 `my-notion`。
- CLI 支持 `auth`、`config`、`docs`、`tokens`、`mcp`、`install`，覆盖初始化检查、文档创建、读取、搜索、列出、更新、导入、导出、归档和 MCP STDIO server。
- Skills 源目录为 `packages/my-notion-skills`，同步到 `.trae/skills` 和 `packages/my-notion-cli/skills`，随 `@mynotion/cli` npm 包发布。
- MCP STDIO 工具包括 `my_notion_docs_search`、`my_notion_docs_fetch`、`my_notion_docs_create`、`my_notion_docs_update`；写工具默认 `dryRun: true`。
- CLI/MCP E2E 已覆盖文档 CRUD、错误契约、PAT 生命周期和 MCP JSON-RPC STDIO 链路。
- 2026-06-01 补齐 `my-notion config init` 首屏初始化入口，集中输出 Node、profile 配置、登录态、Skills 安装提示、MCP 启动命令和下一步建议，并避免输出完整 Token。
- 2026-06-01 补齐 `pnpm e2e:mcp:client` 真实 MCP SDK Client 验证，覆盖工具发现、认证失败、dry-run、确认式真实写入、读取、追加、搜索、归档清理和测试 PAT 撤销。
- 2026-06-01 明确 Agent 文档读写格式策略：对外统一 `contentMarkdown`，服务端负责 Markdown <-> BlockNote blocks 双向转换；普通 Agent 不直接生成或解析 BlockNote JSON。

## 授权与配置

- CLI 登录主链路从手动复制 PAT 升级为浏览器 Device Flow。
- 授权 URL 只包含 `user_code`，不包含敏感 `device_code`。
- Web 授权页接入 Clerk 登录保护、验证码核对、权限清单、i18n、主题切换和 5 秒后返回文档首页提示。
- 线上与本地登录态隔离：线上默认 `~/.local/share/my-notion/config.json`，本地调试 `~/.local/share/my-notion/config.local.json`。
- 默认入口始终是线上 `prod`；本地调试必须显式使用 `--local`。
- 旧 `~/.my-notion` 路径兼容逻辑已移除。

## npm beta 发布

- 首版 npm 包已发布：`@mynotion/cli@0.1.0-beta.0`。
