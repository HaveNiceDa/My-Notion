# CLI 使用说明补充当前项目线上配置

## 目标

补充 `packages/my-notion-cli/docs/usage.md`，让使用者知道在当前线上 My-Notion 项目里从哪里获取 PAT、API URL，以及如何完成首次登录和文档写入。

## 已完成

- 在 `usage.md` 顶部新增“当前项目线上版怎么用”章节。
- 明确当前项目 Machine API 地址：
  - `https://handsome-stoat-500.convex.site`
- 说明 PAT 获取路径：
  - My-Notion Web 登录后打开设置。
  - 在 `API Tokens` 区域创建 token。
  - 勾选 `docs:read` 和 `docs:write`。
  - 复制只显示一次的 `mnt_...` token。
- 说明未来更换 Convex 部署时如何确认 API URL：
  - 优先读取 `NEXT_PUBLIC_CONVEX_SITE_URL`。
  - 如果只有 `NEXT_PUBLIC_CONVEX_URL=https://xxx.convex.cloud`，将 `.convex.cloud` 替换为 `.convex.site`。
