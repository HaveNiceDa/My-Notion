# M14 ReAct Agent Loop 重构

## 完成的改动

- 新增 `lib/agent/tools/definitions.ts`：`AgentTool` 接口 + `knowledgeSearchTool` / `documentReadTool` 定义
- 新增 `lib/agent/tools/registry.ts`：`buildAvailableTools()` 根据上下文构建可用 tool 列表
- 新增 `lib/agent/stream.ts`：从 route.ts 抽出 `AgentStreamEvent` 类型、`enqueueEvent`、`streamModelResponse`、`createThinkingBody`
- 新增 `lib/agent/react-loop.ts`：`runReActLoop` ReAct 循环引擎（MAX_ITERATIONS=5）
- 重写 `route.ts`：精简为 auth → 解析 → buildTools → runReActLoop
- 删除 `knowledge-search.ts` 中的 `KNOWLEDGE_SEARCH_SIGNALS`、`shouldUseKnowledgeSearch()`、`createKnowledgeSearchToolCall()`
- 删除 `document-read.ts` 中的 `DOCUMENT_READ_SIGNALS`、`shouldReadCurrentDocument()`、`createDocumentReadToolCall()`
- 删除 `types.ts` 中的 `PendingToolCall`、`ToolExecutionResult`（改用 OpenAI 原生 `ChatCompletionMessageFunctionToolCall`）
- 移除前端 `useAIChat.ts` 中的 `mode: "auto"` 请求参数
- 更新 `tools/index.ts` 导出
